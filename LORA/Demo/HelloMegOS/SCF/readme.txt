系统：MsgOS-51  
版本：V1.0		 

简介：“MsgOS-51”是一个针对51内核单片机的 协作式单片机系统，它不仅仅只有一个内核，还包括动态内存分配，时间管理，
       日志记录，SHELL交互,等等整个程序框架。本着简单实用的目的，使用MsgOS-51可以使你的应用设计更加高效稳定。
	   ”MsgOS-51“ 是作者MsgOS系列单片机系统的第一个实用型系统，与后期的“MsgOS-M3“相比，虽然很简单，但对于大部分还在用
	   超循环前后台的新手来说还是有很大的先进性的。因为作者比较忙，”MsgOS-51“的说明文档还没有完成，不过整个程序还是
	   比较简单的，代码量也不大，感兴趣的朋友认真分析一下，应该不难懂。云端的高手们看不看意义不大，就不要拍砖了吧。
	   ”MsgOS-51“系统完全开源并免费，但保留版权。欢迎大家学习使用。不足之处一定很多，但还是希望大家能支持一下原创。

	   “MsgOS”即 Message Operating System
